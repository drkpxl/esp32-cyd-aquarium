ESP32 CYD Aquarium

Web Installer for the Cheap Yellow Display

Connect your ESP32-2432S028R (CYD) via USB, then click the button below to install or update the aquarium firmware directly from your browser.

Prerequisites

  • A Chrome or Edge browser (required for Web Serial).
  • An ESP32-2432S028R board connected via USB.
  • Some USB data cables only charge — make sure yours supports data.

Steps

  • Plug the CYD into your computer.
  • Click Install above and select the serial port.
  • Wait for the flash to complete, then your aquarium starts automatically.

Fully Offline

This build is a self-contained aquarium — no clock, no Wi-Fi, and no network setup. Just flash it and watch the fish.
